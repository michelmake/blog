--- layout: home search_exclude: true --- This site is for all information about the project [CORESPONSE](https://coresponse.github.io/blog/). The objective is the development of a simplified ventilation device that is built with the help of a 3D printer and a few components so that it can be easily reproduced and assembled by volunteers all over the world. This device aims to save lives during the COVID19 crisis. ![]({{site.baseurl}}/images/SchemeCORESPONSE_PNG.png "Scheme of the simplified ventilation device") ![]({{site.baseurl}}/images/UKA Logo CMYK.png "UKA Logo") ![]({{site.baseurl}}/images/rwth_irt_bild_rgb.png "IRT Logo") # Posts

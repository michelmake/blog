--- layout: home search_exclude: true --- This site is for all information about the project [CORESPONSE](https://coresponse.github.io/blog/). > **_NOTE:_** Project in progress! The objective is the development of a simplified ventilation device that is built with the help of a 3D printer and a few components so that it can be easily reproduced and assembled by volunteers all over the world. This device aims to save lives during the COVID19 crisis. Follow us on [Youtube](https://www.youtube.com/channel/UC6O9d39XPNeixf8CV6jUgew)! More details on the designs you can find [here](https://www.thingiverse.com/coresponse)! # Posts
